# Car Price Prediction using Machine Learning

## Objective
To build a machine learning model that predicts used car prices.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit Learn
- Streamlit

## Steps
1. Load dataset
2. Data preprocessing
3. Train Linear Regression model
4. Evaluate model
5. Predict car prices

## Outcome
The model predicts car selling prices based on different features.
